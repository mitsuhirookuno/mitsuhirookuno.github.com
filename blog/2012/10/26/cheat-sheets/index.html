チートしようぜ！(ruby Cheat Sheets)

2012年 10月 26日 | Comments

夜中にmochaでのテストの書き方が解らね～とネットを徘徊していると。。。

便利なサイトを発見しました。

mochaチートシート のページです。
チートシート（カンニングペーパー）の言葉のイメージは悪いものの、先駆者が作ってくれた技術Tipsメモって事です。

どうやら、他にも沢山のチートシートがあるらしく
トップページを見て見ることに
あれ・・・？
gemコマンドの説明がある、と言う事は、、、「こいつ・・・動くぞ！」

コマンドとしてインストール可能

$ gem install cheat
Fetching: cheat-1.3.0.gem (100%)
Successfully installed cheat-1.3.0
1 gem installed
Installing ri documentation for cheat-1.3.0...
Installing RDoc documentation for cheat-1.3.0...

なんと、アプリとしてインストールが出来ちゃいます。

使い方は

  • cheatコマンドの使い方
    cheat cheat
  • チートシート一覧を見る
    cheat sheets
  • チートシート一覧からrspecを探す場合
    cheat sheets | grep rspec
  • cheatコマンドの使い方
    cheat sheets | grep rspec
  • mochaのチートシート
    cheat mocha

  • vimのチートシートと出力例を一部
    cheat vim

    Cursor movement

h - move left
j - move down
k - move up
l - move right
ctrl-b - page up
以下、割愛

コピーペーストの文化は、技術的負債に繋がりますが
使い方次第では、とっても便利なものです。
これ日本語化を誰かしてくれないかな～(cheatjみたいな）
